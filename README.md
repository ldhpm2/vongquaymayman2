# Lucky Wheel - Next.js Version

This project has been exported from a Vite-based React app to a Next.js project, ready for Vercel deployment.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This app is ready to be deployed to Vercel. Simply push your code to a GitHub repository and import it into Vercel.

## Features

- **Lucky Wheel**: Smooth spinning animation with physics-based easing.
- **Excel Support**: Import student lists directly from Excel files.
- **Score Management**: Track and edit scores for up to 5 categories.
- **Persistence**: Data is saved to `localStorage` for convenience.
- **Export**: Download your updated score table back to Excel.
